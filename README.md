# FTS
challareadyknow
## Setup
Install Ruby 2.3.0
https://www.ruby-lang.org/en/documentation/installation/

Install Ruby Gems
https://rubygems.org/pages/download

Install bundler
`gem install bundler`


### Running the project
cd to the root directory of the project

Get the gems using `bundle install`

Run the project using `ruby main.rb`
